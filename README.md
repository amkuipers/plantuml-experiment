# plantuml-experiment
Experiment with PlantUML


## Example with embedded

Code block with PlantUML. Adding a GitHub Action to generate the image; this is not needed using mermaid.js or on gitlab.

```plantUML
@startuml
Alice -> Bob: test
@enduml
```
